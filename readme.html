SoudBoard

The goal of this project is make a multifunction painel control(like stream deck). Compatible to a DIY "SoundBoard" utilizing ESP32

Features

  • Soundpad - Send any local mp3 sound effect you like. And link with your SoundBoard
  • Control Software - Dynamicilly link with Softwares like OBS, Spotfy, Photoshop, Microsoft Office, Open URLs, and assign keybinds
  • Workflow Automation - Assing mutiple actions, like muting your microfone, set brightness, set volume, switch cameras, lauching apps, all to a single button

    • Software:

  • Python3.14.3
  • pyserial
  • dearpygui
  • playsound3

    • Hardware:

  • ESP32-C3 SuperMini
  • 9x Push Button
