ESP32 Multifunction Control Panel

Overview

A multifunction desktop control panel inspired by devices like the Stream Deck, built using an ESP32 and Python. Originally designed as a DIY soundboard, the project evolved into a programmable productivity and workflow automation tool capable of controlling applications, executing macros, managing media, and triggering custom actions through physical buttons. The system combines hardware and software integration to provide a low-cost and customizable control interface for streamers, productivity workflows, and multimedia application
